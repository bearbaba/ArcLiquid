# ArcLiquid

Permissionless Stablecoin Money Market on Arc L1 (Circle)

## Main Features
- Supply & Borrow USDC (native stablecoin)
- Dynamic interest rate model (based on utilization)
- Opt-in Compliance (KYC mock)
- Beautiful dark mode frontend

## Quick Start
1. Claim test USDC at https://faucet.circle.com
2. Deploy contract with Foundry
3. Run frontend

## Tech Stack
- Solidity + Foundry (contracts)
- React + Vite + wagmi/viem (frontend)
- Arc Testnet (Chain ID 5042002)
